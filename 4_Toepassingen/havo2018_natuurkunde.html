Algemene Uitwerking Vraag1¶

Bestudeer de Algemene Uitwerking van Vraag1

import Html exposing(..)
main = text ("Afstand = " ++ (toString afstand) ++ " m")

c = 3.0*(10^8)
milli g = g * 10^(-3)

s_signaal t = c * t
s_object t = 0.5 * (s_signaal t)

afstand = s_object (milli 0.26)
